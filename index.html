Aim Training Mod


Visit Steam Workshop SiteGithub







• Modified faction to hold all inventory items to easy access

• Flick training range

• Shooting ranges that have automatically moving, damageable soldiers at consistent speeds.

• Eliminated soldiers are revived instantly, have a sound indicator when hit

• 2 CQB sites which include custom modified AI soldiers.

• Sites with pop up dummies (with in-game buildings)

AI modifications were mostly done for optimization reasons but added a little bit of custom logic to change the time for aim and time to kill

Capture the Flag Game Mode


Visit Steam Workshop SiteGithub



A Gamemode inspired from Call of Duty 2. Multiplayer ready fully replicated, including pop-up UI notifications. None of the blueprints uses Squad game-related logic so it is ready to use for all Unreal Engine versions. It has the same attributes as Call of Duty 2 such as;

Carrying a flag

Dropping a flag from enemy

Returning flag to base

Capturing a flag (both flags should be on the same base)

When a flag is dropped team symbol appears on the position of the flag and disappears when the flag is taken

all of the listed actions have distinct sounds and UI messages

it could be easily modified to multiple flags and multiple flag bases version

Paratrooper Command Call




Made changes at vanilla airstrike command call to make it usable for active spawn point. When the Team commander chooses paratrooper action a transport plane appears on the map and for a brief moment, players can select a spawn point as a paratrooper action and jump from the plane the jumping action is unique thing is about this action because it does not use cinematics or seated players to take action. Players can walk inside the plane they are not pushing players or blocking each other and they can jump from the tail of the plane. Jumped players are open parachutes at a fixed altitude and can land at steered points.

Fast Insertion Rope Extraction System


Non-attack helicopters have FIRES system to able users to jump from helicopters and land safely and fast pace. Only seat numbers 5 and 8 have this system so players have to coordinate and realistically wait for each other. Helicopter pilots can drop the ropes at less than 20 meters altitude and low speed by using a landing camera functionality. Ropes drop down from socket to ground via UE4 cable actor




Admin Cam Panel




For easy use for admin tools

• Warn, Kick, Ban players from the server these functionalities come from squad SDK however unique thing about this panel is admins are able to choose players at runtime without typing or scrolling a list they just have to aim and shoot with a small crosshair to select players.

• The Admin panel can also spawn vehicles at the current position of the admin camera. The vehicle list can be changed from the data table so other team members don't have to deal with hardcoded variables of Squad SDK.

• Lastly, the admin panel can also change level to day or night via a single button

This panel gives freedom to server owners/admins, easy access to base and custom game settings

Admin panel can be minimized and removed completely from the screen. All of the listed functionalities are replicated.

Punji Trap


• Has a marker on the minimap as a mine

• Camouflaged trap for Vietcong forces which activates when the enemy team member steps on it

• The first victim gets naturalized

• Second victim gets %70 damage and starts to bleed and gets movement penalized for 5 seconds

• Third victim gets %50 damage and starts to bleed and gets movement penalized for 5 seconds

• Other victims for the same trap get the same treatment as 3rd player

• Disappears from the map and minimap after 60 seconds after the first victim

• Or disappears from map 40 seconds and 20 seconds for 2nd and 3rd victim
